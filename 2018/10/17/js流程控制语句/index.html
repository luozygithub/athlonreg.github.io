3、循环机构

循环：重复去做一件事情

for循环

格式：

for(表达式1；表达式2；表达式3){

​ 循环的代码；

}

执行过程：

​ 1、先会去执行表达式1，再去判断表达式2，如果表达式为 true，执行循环代码。如果表达式为false，结束循环。

​ 2、再去执行表达式3，判断表达式2是否成立，重复1,2。

ps:好处：必要表达式不容易丢；

while循环

循环格式：

​ while（循环条件）{

​ 如果条件成立，运行此处代码；

​ }

循环编写过程：

1、确定循环次数

2、每次执行的动作

ps: 注意不要造成死循环

do while循环

与while基本相同

唯一区别：do while 第一次会先执行再去判断。

【注】：

do{

}while();

ps: do while 几乎不用 ， while循环用的比较少， for循环经常使用
